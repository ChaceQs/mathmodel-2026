## 图A：问题二 MILP 模型流程图
**用途**：展示调度优化模型的完整求解流程，放在论文"模型建立"小节。

**输入数据（3项）**：
站点基础信息、距离矩阵 / 每日初始库存、期望库存 / 天气数据、是否工作日

**数据预处理步骤**：
计算每个站点的库存偏差（初始库存 - 期望库存），分类为供给站点（积压）和需求站点（缺货），匹配当天是否为工作日以选择对应的期望库存

**模型核心公式**：

决策变量（3个）：
- `x(k,i,j)` — 卡车k从站点i到站点j的调度量，整数，0到15辆
- `y(k,i,j)` — 是否执行该调度，0或1
- `I(i)` — 站点i调度后的库存

目标函数：
- min Z = 运输成本 + 缺货成本 + 积压成本
- 运输成本 = Σ x(k,i,j) × 距离(i,j) × 2元/km
- 缺货成本 = Σ max(0, 期望库存下限 - I(i)) × 10元/辆
- 积压成本 = Σ max(0, I(i) - 期望库存上限) × 5元/辆

约束条件（5类）：
- 库存平衡：I(i) = 初始库存(i) + Σ调入 - Σ调出
- 车辆容量：x(k,i,j) ≤ 15辆
- 时间限制：Σ(行驶时间 + 装卸时间) ≤ 300分钟（00:00-05:00）
- 供需可行性：总调出量 ≤ 站点当前库存
- 逻辑约束：x(k,i,j) ≥ 0 且为整数

**求解**：
CBC求解器（PuLP库），求解状态 Optimal，求解时间 0.01秒，最优值 242.20元

**输出结果（3项）**：
调度方案表（8次调度，24辆） / 成本效益分析（总成本355→242.20元，降低31.8%） / 多日期验证（6月1-14日共14天全部求得最优解）

## 图B：问题三 MPC 滚动时域框架图
**用途**：展示预测-调度一体化的闭环滚动机制，放在论文"滚动时域框架"小节。

**前置**：
使用6月1-10日共10天数据，训练XGBoost模型（26个特征，MAE=0.29辆，MAPE=1.04%，R²=0.998）

**核心循环（四步，每天重复执行）**：

步骤1 — 预测需求：基于当前最新的历史库存数据，用XGBoost模型预测当天的各站点期望库存（10个站点各一个预测值）

步骤2 — MILP优化：将预测的期望值输入问题二的MILP调度模型，求解当日最优调度方案（哪辆卡车从哪个站点到哪个站点调多少辆）

步骤3 — 执行调度：按照求解出的方案，卡车将车辆从过剩站点运往短缺站点

步骤4 — 观测库存：调度完成后，获取各站点实际库存数据，将其反馈到历史数据集中，更新模型输入状态

然后返回步骤1，滚动执行下一天。

**时间范围**：覆盖6月11日-14日共4天（题目要求2天，超额完成）

**反馈回路关键**：第12天成本从112元骤升至659元，正是因为第11天调度执行后出现了新的供需缺口，MPC框架自动感知并触发大规模调度响应——体现了滚动时域的自适应性和鲁棒性。

**整体结果**：4天总成本1,422.76元，日均355.69元。

## 图C：整体技术路线图
**用途**：展示三个问题之间的递进关系，放在论文"问题分析"末尾。

**问题一：数据特征分析与站点分类**
- 分析方法：缺失值检查、库存-骑行-调度一致性验证、潮汐指数计算（|早高峰净流出|+|晚高峰净流入|）、K-means聚类（3类，轮廓系数0.693）、关键度评分（0.3×流出+0.3×流入+0.2×潮汐+0.2×|净调度|）、天气/节假日影响分析
- 输出：3类站点分类（供给型/需求型/平衡型）、10个站点关键度排名

**问题二：最优控制调度优化**
- 核心模型：MILP（混合整数线性规划）
- 约束：3辆卡车、容量15辆/车、300分钟时间窗口、20km/h、装卸1分钟/辆
- 成本结构：运输2元/km·辆、缺货10元/辆·次、积压5元/辆·次
- 对照方法：贪心启发式算法
- 验证：14天全量多日期验证
- 结果：成本降低31.8%（355→242.20元），需求满足率30%→80%，平衡度改善46.5%

**问题三：需求预测与集成应用**
- 预测模型：XGBoost（主）+ 卡尔曼滤波（辅，作为微分方程对照）
- 精度：MAE=0.29辆，MAPE=1.04%，R²=0.998
- 集成：预测需求输入MILP → 调度优化 → 对比历史平均需求（成本降低37.9%）
- MPC框架：4天滚动时域预测-调度一体化，提升系统鲁棒性

**问题递进关系**：
问题一 → 问题二：问题一的站点分类和关键度评分为问题二提供调度优先级参考
问题二 → 问题三：问题二的MILP调度模型作为问题三预测后端的执行引擎

**底部汇总**：成本降低31.8% | 满足率80%+ | MAPE 1.04% | 14天全通过
