# 问题二：共享单车调度优化的统计建模方案

## 一、引言

从应用统计学视角，问题二本质上是一个**数据驱动的决策优化问题**。与纯数学优化不同，统计建模方案强调：
1. 基于历史数据的参数估计
2. 考虑不确定性和随机性
3. 模型的统计显著性检验
4. 预测精度的量化评估

**实际数据基础**：
- **数据规模**：2023年6月1日-6月14日，共14天数据
- **站点数量**：10个站点（S001-S010）
- **历史调度**：61条调度记录
- **特征维度**：约17维特征（站点属性、时间特征、天气、历史统计）
- **目标变量**：调度量、调度方向、库存偏差
- **样本量**：14天×10站点=140个观测（满足统计建模最小样本量要求）

本文档设计完整的统计建模方案，包括数据预处理、特征工程、模型选择、参数估计、假设检验等环节。

## 二、数据预处理策略

### 2.1 数据质量评估

#### 2.1.1 完整性检查
```python
# 缺失值检测
缺失率 = (缺失值数量 / 总样本数) × 100%
```

**处理准则**：
- 缺失率 < 5%：删除缺失样本
- 5% ≤ 缺失率 < 20%：插补处理
- 缺失率 ≥ 20%：考虑删除该变量或使用高级插补方法

**插补方法选择**：
| 变量类型 | 推荐方法 | 理由 |
|---------|---------|------|
| 连续型（正态分布） | 均值插补 | 保持均值不变 |
| 连续型（偏态分布） | 中位数插补 | 抗异常值 |
| 分类型 | 众数插补 | 保持分布 |
| 时间序列 | 线性插值/ARIMA | 保持趋势 |
| 多变量相关 | KNN插补/MICE | 利用变量间关系 |

#### 2.1.2 异常值检测

**方法1：箱线图法（IQR）**
```
Q1 = 第25百分位数
Q3 = 第75百分位数
IQR = Q3 - Q1
异常值：x < Q1 - 1.5×IQR 或 x > Q3 + 1.5×IQR
```

**方法2：Z-score法**
```
Z = (x - μ) / σ
异常值：|Z| > 3
```

**方法3：Isolation Forest（适用于高维数据）**
- 基于随机森林的无监督异常检测
- 异常样本更容易被隔离

**处理策略**：
1. **保留**：如果异常值有实际意义（如节假日、特殊事件）
2. **删除**：如果是数据录入错误
3. **截尾**：替换为上下界值
4. **变换**：对数变换、Box-Cox变换降低影响

#### 2.1.3 一致性检验

**逻辑一致性**：
```python
# 库存守恒检验
次日库存 = 当日库存 + 净流入 + 净调度
残差 = |实际次日库存 - 理论次日库存|
一致性指标 = (残差 ≤ 阈值的样本数) / 总样本数
```

**时间一致性**：
- 检查时间戳的连续性
- 识别重复记录
- 验证时间顺序

### 2.2 数据标准化与归一化

#### 2.2.1 标准化（Z-score Standardization）
$$x_{std} = \frac{x - \mu}{\sigma}$$

**适用场景**：
- 变量服从正态分布或近似正态
- 需要保留异常值信息
- 用于线性回归、PCA等

#### 2.2.2 归一化（Min-Max Normalization）
$$x_{norm} = \frac{x - x_{min}}{x_{max} - x_{min}}$$

**适用场景**：
- 变量有明确的上下界
- 需要将数据映射到[0,1]区间
- 用于神经网络、距离计算等

#### 2.2.3 鲁棒标准化（Robust Scaling）
$$x_{robust} = \frac{x - \text{median}}{IQR}$$

**适用场景**：
- 数据包含异常值
- 分布偏态严重

### 2.3 样本大小评估

#### 2.3.1 最小样本量计算

**回归分析**：
$$n \geq 10k + 50$$
其中 $k$ 为自变量个数

**本问题**：
- 自变量数：约10-15个（站点特征、时间特征、天气特征等）
- 最小样本量：$n \geq 10 \times 15 + 50 = 200$
- 实际样本量：30天 × 10站点 = 300个观测，**满足要求**

#### 2.3.2 统计功效分析（Power Analysis）

**目标**：
- 显著性水平 $\alpha = 0.05$
- 统计功效 $1-\beta = 0.80$
- 效应量 $d = 0.5$（中等效应）

**结论**：当前样本量足够检测中等及以上效应

### 2.4 数据分割策略

#### 2.4.1 时间序列分割（推荐）
```
训练集：前21天（70%）
验证集：中间6天（20%）
测试集：最后3天（10%）
```

**理由**：
- 保持时间顺序，避免数据泄露
- 符合实际应用场景（用历史预测未来）

#### 2.4.2 交叉验证策略

**时间序列交叉验证（Time Series CV）**：
```
Fold 1: Train[1-14] → Test[15-17]
Fold 2: Train[1-17] → Test[18-20]
Fold 3: Train[1-20] → Test[21-23]
Fold 4: Train[1-23] → Test[24-26]
Fold 5: Train[1-26] → Test[27-30]
```

**优势**：
- 充分利用数据
- 评估模型稳定性
- 避免过拟合

## 三、特征工程方法

### 3.1 特征类型划分

#### 3.1.1 原始特征（直接从数据提取）

**站点特征**：
- 容量（Capacity）
- 位置坐标（Latitude, Longitude）
- 站点类型（商业区、住宅区、交通枢纽等）

**时间特征**：
- 日期（Date）
- 星期几（Weekday）
- 是否工作日（IsWorkday）
- 是否节假日（IsHoliday）

**天气特征**：
- 温度（Temperature）
- 降水量（Precipitation）
- 风速（WindSpeed）
- 天气类型（WeatherType）

**流量特征**：
- 流入量（Inflow）
- 流出量（Outflow）
- 净流量（NetFlow = Inflow - Outflow）

**库存特征**：
- 初始库存（InitialInventory）
- 期望库存（TargetInventory）
- 库存偏差（InventoryDeviation）

#### 3.1.2 衍生特征（通过计算生成）

**统计特征**：
```python
# 滚动统计
流入量_7日均值 = rolling_mean(Inflow, window=7)
流入量_7日标准差 = rolling_std(Inflow, window=7)
流入量_7日最大值 = rolling_max(Inflow, window=7)
流入量_7日最小值 = rolling_min(Inflow, window=7)
```

**滞后特征**：
```python
# 时间滞后
流入量_lag1 = Inflow.shift(1)  # 前1天
流入量_lag7 = Inflow.shift(7)  # 前7天（上周同期）
```

**交互特征**：
```python
# 特征交叉
温度×工作日 = Temperature × IsWorkday
流入量×容量比 = Inflow / Capacity
```

**时间编码**：
```python
# 周期性编码（避免断点问题）
星期_sin = sin(2π × Weekday / 7)
星期_cos = cos(2π × Weekday / 7)
```

### 3.2 特征选择方法

#### 3.2.1 过滤法（Filter Methods）

**方法1：相关系数法**
```python
# Pearson相关系数
r = corr(X, y)
选择：|r| > 阈值（如0.3）
```

**方法2：方差分析（ANOVA F-test）**
```python
# F统计量
F = MSB / MSW
p_value = F_test(F, df1, df2)
选择：p_value < 0.05
```

**方法3：互信息（Mutual Information）**
```python
# 衡量变量间的依赖关系
MI(X, Y) = ∑∑ p(x,y) log(p(x,y) / (p(x)p(y)))
选择：MI > 阈值
```

#### 3.2.2 包裹法（Wrapper Methods）

**方法1：递归特征消除（RFE）**
```
1. 训练模型，计算特征重要性
2. 删除最不重要的特征
3. 重复步骤1-2，直到达到目标特征数
```

**方法2：前向选择（Forward Selection）**
```
1. 从空集开始
2. 每次添加一个使模型性能提升最大的特征
3. 重复直到性能不再提升
```

**方法3：后向消除（Backward Elimination）**
```
1. 从全集开始
2. 每次删除一个对模型性能影响最小的特征
3. 重复直到性能开始下降
```

#### 3.2.3 嵌入法（Embedded Methods）

**方法1：Lasso回归（L1正则化）**
$$\min_{\beta} \sum_{i=1}^n (y_i - X_i\beta)^2 + \lambda \sum_{j=1}^p |\beta_j|$$

**特点**：
- 自动进行特征选择（部分系数压缩为0）
- 适合高维数据

**方法2：随机森林特征重要性**
```python
# 基于Gini不纯度或信息增益
importance = RandomForest.feature_importances_
选择：importance > 阈值
```

**方法3：梯度提升树（XGBoost/LightGBM）**
```python
# 基于增益、覆盖度、频率
importance = XGBoost.get_score(importance_type='gain')
```

### 3.3 特征变换

#### 3.3.1 非线性变换

**对数变换**（处理右偏分布）：
$$x' = \log(x + 1)$$

**Box-Cox变换**（使数据更接近正态分布）：
$$x' = \begin{cases}
\frac{x^\lambda - 1}{\lambda} & \lambda \neq 0 \\
\log(x) & \lambda = 0
\end{cases}$$

**Yeo-Johnson变换**（可处理负值）：
$$x' = \begin{cases}
\frac{(x+1)^\lambda - 1}{\lambda} & \lambda \neq 0, x \geq 0 \\
\log(x+1) & \lambda = 0, x \geq 0 \\
-\frac{(-x+1)^{2-\lambda} - 1}{2-\lambda} & \lambda \neq 2, x < 0 \\
-\log(-x+1) & \lambda = 2, x < 0
\end{cases}$$

#### 3.3.2 降维方法

**主成分分析（PCA）**：
$$Z = XW$$
其中 $W$ 是特征向量矩阵

**选择主成分数量**：
- 累计方差贡献率 ≥ 85%
- 碎石图（Scree Plot）拐点

**因子分析（Factor Analysis）**：
$$X = LF + \epsilon$$
其中 $L$ 是因子载荷矩阵，$F$ 是公共因子

### 3.4 特征重要性评估

#### 3.4.1 统计显著性

**t检验**（单个特征）：
$$t = \frac{\hat{\beta}_j}{SE(\hat{\beta}_j)}$$

**F检验**（整体显著性）：
$$F = \frac{(RSS_0 - RSS_1) / p}{RSS_1 / (n-p-1)}$$

#### 3.4.2 预测贡献度

**SHAP值（SHapley Additive exPlanations）**：
$$\phi_j = \sum_{S \subseteq N \setminus \{j\}} \frac{|S|!(|N|-|S|-1)!}{|N|!} [f_{S \cup \{j\}}(x_{S \cup \{j\}}) - f_S(x_S)]$$

**优势**：
- 理论基础扎实（博弈论）
- 可解释性强
- 适用于任何模型

#### 3.4.3 稳定性评估

**Bootstrap重采样**：
```
1. 重复B次（如1000次）：
   a. 有放回抽样
   b. 训练模型
   c. 记录特征重要性
2. 计算重要性的均值和置信区间
3. 选择置信区间不包含0的特征
```

### 3.5 针对本问题的特征工程方案

#### 3.5.1 核心特征集

**必选特征**（基于问题一的分析）：

## 四、统计模型设计

### 4.1 模型选择框架

#### 4.1.1 问题分解

问题二可分解为三个统计建模子问题：

**子问题1：需求预测模型**
- 目标：预测各站点的期望库存 $I_i^*$
- 类型：回归问题
- 输入：站点特征、时间特征、天气特征
- 输出：期望库存（连续值）

**子问题2：调度量预测模型**
- 目标：预测站点对$(i,j)$之间的最优调度量 $x_{ij}$
- 类型：回归问题（可转化为分类问题）
- 输入：站点对特征、库存偏差、距离
- 输出：调度量（非负整数）

**子问题3：调度优先级分类模型**
- 目标：识别高优先级调度站点对
- 类型：分类问题
- 输入：站点对特征
- 输出：优先级类别（高/中/低）

### 4.2 候选统计模型

#### 4.2.1 线性回归模型族

**模型1：多元线性回归（MLR）**
$$y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_p x_p + \epsilon$$

**优势**：
- 可解释性强
- 计算简单
- 参数估计稳定

**劣势**：
- 假设线性关系
- 对异常值敏感
- 不能捕捉非线性

**模型2：岭回归（Ridge Regression）**
$$\min_{\beta} \sum_{i=1}^n (y_i - X_i\beta)^2 + \lambda \sum_{j=1}^p \beta_j^2$$

**优势**：
- 处理多重共线性
- 防止过拟合
- 所有特征保留

**模型3：Lasso回归**
$$\min_{\beta} \sum_{i=1}^n (y_i - X_i\beta)^2 + \lambda \sum_{j=1}^p |\beta_j|$$

**优势**：
- 自动特征选择
- 稀疏解
- 适合高维数据

**模型4：弹性网络（Elastic Net）**
$$\min_{\beta} \sum_{i=1}^n (y_i - X_i\beta)^2 + \lambda_1 \sum_{j=1}^p |\beta_j| + \lambda_2 \sum_{j=1}^p \beta_j^2$$

**优势**：
- 结合Lasso和Ridge优点
- 处理相关特征组
- 稳定性好

#### 4.2.2 广义线性模型（GLM）

**泊松回归**（适用于计数数据）：
$$\log(\mu) = \beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p$$
$$Y \sim \text{Poisson}(\mu)$$

**适用场景**：
- 调度量预测（非负整数）
- 流量预测

**负二项回归**（处理过度离散）：
$$Y \sim \text{NB}(\mu, \theta)$$

**适用场景**：
- 方差 > 均值的计数数据
- 更灵活的分布假设

#### 4.2.3 树模型族

**决策树回归（CART）**：
$$\hat{y} = \sum_{m=1}^M c_m \mathbb{1}(x \in R_m)$$

**随机森林（Random Forest）**：
$$\hat{y} = \frac{1}{B} \sum_{b=1}^B T_b(x)$$

**优势**：
- 非线性关系
- 特征交互自动捕捉
- 鲁棒性强
- 特征重要性评估

**梯度提升树（Gradient Boosting）**：
$$F_m(x) = F_{m-1}(x) + \gamma_m h_m(x)$$

**XGBoost/LightGBM**：
- 高效实现
- 正则化防止过拟合
- 处理缺失值
- 特征重要性

#### 4.2.4 时间序列模型

**ARIMA模型**：
$$\phi(B)(1-B)^d y_t = \theta(B)\epsilon_t$$

**适用场景**：
- 单变量时间序列预测
- 短期预测

**VAR模型（向量自回归）**：
$$\mathbf{y}_t = \mathbf{c} + \mathbf{A}_1 \mathbf{y}_{t-1} + \cdots + \mathbf{A}_p \mathbf{y}_{t-p} + \mathbf{\epsilon}_t$$

**适用场景**：
- 多站点联合预测
- 捕捉站点间相互影响

### 4.3 模型选择准则

#### 4.3.1 信息准则

**AIC（Akaike Information Criterion）**：
$$AIC = 2k - 2\ln(\hat{L})$$

**BIC（Bayesian Information Criterion）**：
$$BIC = k\ln(n) - 2\ln(\hat{L})$$

**选择规则**：AIC/BIC越小越好

**对比**：
- BIC对模型复杂度惩罚更重
- 样本量大时，BIC倾向选择更简单模型

#### 4.3.2 交叉验证准则

**K折交叉验证**：
$$CV = \frac{1}{K} \sum_{k=1}^K MSE_k$$

**时间序列交叉验证**：
- 保持时间顺序
- 滚动窗口验证

#### 4.3.3 预测性能准则

**回归问题**：
- $R^2$（决定系数）：$R^2 = 1 - \frac{SS_{res}}{SS_{tot}}$
- RMSE（均方根误差）：$RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^n (y_i - \hat{y}_i)^2}$
- MAE（平均绝对误差）：$MAE = \frac{1}{n}\sum_{i=1}^n |y_i - \hat{y}_i|$
- MAPE（平均绝对百分比误差）：$MAPE = \frac{100\%}{n}\sum_{i=1}^n |\frac{y_i - \hat{y}_i|}{y_i}|$

**分类问题**：
- 准确率（Accuracy）
- 精确率（Precision）
- 召回率（Recall）
- F1-score
- AUC-ROC

### 4.4 推荐模型方案

#### 4.4.1 基准模型（Baseline）

**简单线性回归**：
- 快速实现
- 建立性能基准
- 可解释性强

#### 4.4.2 主推模型

**梯度提升树（XGBoost/LightGBM）**

**推荐理由**：
1. 预测精度高
2. 自动处理非线性和交互
3. 鲁棒性强（对异常值不敏感）
4. 特征重要性评估
5. 适合中小规模数据

**模型配置**：
```python
XGBoost参数：
- objective: 'reg:squarederror'（回归）
- max_depth: 5-7
- learning_rate: 0.01-0.1
- n_estimators: 100-500
- subsample: 0.8
- colsample_bytree: 0.8
- reg_alpha: 0.1（L1正则化）
- reg_lambda: 1.0（L2正则化）
```

#### 4.4.3 备选模型

**随机森林**：
- 作为对照模型
- 评估模型稳定性

**弹性网络**：
- 提供线性基准
- 系数可解释性

## 五、参数估计方法

### 5.1 最小二乘估计（OLS）

#### 5.1.1 理论基础

**目标函数**：
$$\min_{\beta} \sum_{i=1}^n (y_i - X_i\beta)^2$$

**正规方程**：
$$\hat{\beta} = (X^TX)^{-1}X^Ty$$

**性质**：
- 无偏性：$E(\hat{\beta}) = \beta$
- 最小方差（BLUE）
- 一致性：$\hat{\beta} \xrightarrow{p} \beta$

#### 5.1.2 假设条件

1. **线性性**：$E(y|X) = X\beta$
2. **独立性**：观测独立
3. **同方差性**：$\text{Var}(\epsilon_i) = \sigma^2$
4. **正态性**：$\epsilon \sim N(0, \sigma^2)$
5. **无多重共线性**：$X^TX$可逆

#### 5.1.3 标准误估计

$$SE(\hat{\beta}_j) = \sqrt{\hat{\sigma}^2 (X^TX)^{-1}_{jj}}$$

其中 $\hat{\sigma}^2 = \frac{1}{n-p-1}\sum_{i=1}^n (y_i - \hat{y}_i)^2$

### 5.2 最大似然估计（MLE）

#### 5.2.1 理论基础

**似然函数**：
$$L(\theta; y) = \prod_{i=1}^n f(y_i; \theta)$$

**对数似然**：
$$\ell(\theta) = \sum_{i=1}^n \log f(y_i; \theta)$$

**估计量**：
$$\hat{\theta}_{MLE} = \arg\max_{\theta} \ell(\theta)$$

#### 5.2.2 泊松回归MLE

**似然函数**：
$$L(\beta) = \prod_{i=1}^n \frac{e^{-\mu_i} \mu_i^{y_i}}{y_i!}$$

其中 $\log(\mu_i) = X_i\beta$

**求解**：迭代加权最小二乘（IRLS）

### 5.3 正则化估计

#### 5.3.1 岭回归估计

$$\hat{\beta}_{ridge} = (X^TX + \lambda I)^{-1}X^Ty$$

**超参数选择**：
- 交叉验证
- 岭迹图（Ridge Trace）

#### 5.3.2 Lasso估计

**坐标下降算法**：
```
1. 初始化 β = 0
2. for j = 1 to p:
3.     计算偏残差 r = y - X_{-j}β_{-j}
4.     更新 β_j = S(X_j^T r / n, λ) / (X_j^T X_j / n)
5. 重复步骤2-4直到收敛
```

其中 $S(z, \gamma) = \text{sign}(z)(|z| - \gamma)_+$ 是软阈值算子

#### 5.3.3 弹性网络估计

**优化问题**：
$$\min_{\beta} \frac{1}{2n}\sum_{i=1}^n (y_i - X_i\beta)^2 + \lambda[\alpha||\beta||_1 + (1-\alpha)||\beta||_2^2]$$

**求解**：坐标下降 + 路径算法

### 5.4 树模型参数估计

#### 5.4.1 CART分裂准则

**回归树**：
$$\min_{j,s} \left[\min_{c_1} \sum_{x_i \in R_1(j,s)} (y_i - c_1)^2 + \min_{c_2} \sum_{x_i \in R_2(j,s)} (y_i - c_2)^2\right]$$

#### 5.4.2 梯度提升

**算法**：
```
1. 初始化 F_0(x) = arg min_γ Σ L(y_i, γ)
2. for m = 1 to M:
3.     计算伪残差 r_im = -[∂L(y_i, F(x_i))/∂F(x_i)]_{F=F_{m-1}}
4.     拟合回归树 h_m(x) 到 {(x_i, r_im)}
5.     计算步长 γ_m = arg min_γ Σ L(y_i, F_{m-1}(x_i) + γh_m(x_i))
6.     更新 F_m(x) = F_{m-1}(x) + γ_m h_m(x)
```

### 5.5 参数不确定性量化

#### 5.5.1 Bootstrap方法

**非参数Bootstrap**：
```
1. 重复B次（如1000次）：
   a. 有放回抽样得到样本 {(x_i*, y_i*)}
   b. 估计参数 β*
2. 计算Bootstrap标准误：SE_boot = sd(β*)
3. 计算置信区间：[β*_{α/2}, β*_{1-α/2}]
```

#### 5.5.2 贝叶斯方法

**后验分布**：
$$p(\beta|y) \propto p(y|\beta)p(\beta)$$

**MCMC采样**：
- Gibbs采样
- Metropolis-Hastings算法

**优势**：
- 完整的不确定性量化
- 先验信息融合
- 预测区间

### 5.6 针对本问题的参数估计策略

## 六、假设检验程序

### 6.1 回归系数显著性检验

#### 6.1.1 单个系数t检验

**原假设**：$H_0: \beta_j = 0$
**备择假设**：$H_1: \beta_j \neq 0$

**检验统计量**：
$$t = \frac{\hat{\beta}_j}{SE(\hat{\beta}_j)} \sim t(n-p-1)$$

**决策规则**：
- 若 $|t| > t_{\alpha/2}(n-p-1)$，拒绝$H_0$
- 或 p-value < $\alpha$（通常$\alpha=0.05$）

#### 6.1.2 整体显著性F检验

**原假设**：$H_0: \beta_1 = \beta_2 = \cdots = \beta_p = 0$

**检验统计量**：
$$F = \frac{(TSS - RSS)/p}{RSS/(n-p-1)} \sim F(p, n-p-1)$$

**决策规则**：若 $F > F_{\alpha}(p, n-p-1)$，拒绝$H_0$

### 6.2 模型假设检验

#### 6.2.1 正态性检验

**Shapiro-Wilk检验**：
$$W = \frac{(\sum_{i=1}^n a_i x_{(i)})^2}{\sum_{i=1}^n (x_i - \bar{x})^2}$$

**Kolmogorov-Smirnov检验**：
$$D = \sup_x |F_n(x) - F_0(x)|$$

**Q-Q图**：视觉检验

#### 6.2.2 同方差性检验

**Breusch-Pagan检验**：
$$LM = n \cdot R^2_{aux} \sim \chi^2(p)$$

**White检验**（更一般）：
- 允许异方差形式未知

**处理方法**：
- 加权最小二乘（WLS）
- 稳健标准误（Robust SE）

#### 6.2.3 独立性检验

**Durbin-Watson检验**：
$$DW = \frac{\sum_{t=2}^n (e_t - e_{t-1})^2}{\sum_{t=1}^n e_t^2}$$

**判断**：
- DW ≈ 2：无自相关
- DW < 2：正自相关
- DW > 2：负自相关

### 6.3 预测精度检验

#### 6.3.1 Diebold-Mariano检验

**比较两个模型的预测精度**：

**原假设**：$H_0: E[L(e_{1t})] = E[L(e_{2t})]$

**检验统计量**：
$$DM = \frac{\bar{d}}{\sqrt{\text{Var}(\bar{d})}} \sim N(0,1)$$

其中 $d_t = L(e_{1t}) - L(e_{2t})$

### 6.4 多重比较校正

**Bonferroni校正**：
$$\alpha_{adj} = \frac{\alpha}{m}$$

**FDR控制**（Benjamini-Hochberg）：
- 控制错误发现率
- 适合大规模检验

## 七、模型诊断与验证

### 7.1 残差分析

#### 7.1.1 残差图

**标准化残差**：
$$r_i = \frac{e_i}{\hat{\sigma}\sqrt{1-h_{ii}}}$$

**学生化残差**：
$$t_i = \frac{e_i}{\hat{\sigma}_{(i)}\sqrt{1-h_{ii}}}$$

**诊断图**：
1. 残差vs拟合值图（检查同方差性）
2. 残差vs自变量图（检查线性性）
3. Q-Q图（检查正态性）
4. 残差vs时间图（检查独立性）

#### 7.1.2 影响点诊断

**Cook距离**：
$$D_i = \frac{(\hat{y} - \hat{y}_{(i)})^T(\hat{y} - \hat{y}_{(i)})}{p\hat{\sigma}^2}$$

**判断**：$D_i > 1$ 为强影响点

**DFFITS**：
$$DFFITS_i = \frac{\hat{y}_i - \hat{y}_{(i)}}{\hat{\sigma}_{(i)}\sqrt{h_{ii}}}$$

### 7.2 多重共线性诊断

#### 7.2.1 方差膨胀因子（VIF）

$$VIF_j = \frac{1}{1-R_j^2}$$

**判断标准**：
- VIF < 5：无严重共线性
- 5 ≤ VIF < 10：中度共线性
- VIF ≥ 10：严重共线性

**处理方法**：
1. 删除高VIF变量
2. 主成分回归（PCR）
3. 偏最小二乘（PLS）
4. 岭回归/Lasso

#### 7.2.2 条件数（Condition Number）

$$\kappa = \sqrt{\frac{\lambda_{max}}{\lambda_{min}}}$$

**判断**：$\kappa > 30$ 表示严重共线性

### 7.3 过拟合检测

#### 7.3.1 学习曲线

**绘制**：
- 训练误差 vs 样本量
- 验证误差 vs 样本量

**诊断**：
- 高偏差：训练和验证误差都高
- 高方差：训练误差低，验证误差高

#### 7.3.2 正则化路径

**Lasso路径图**：
- 观察系数随$\lambda$变化
- 选择合适的正则化强度

### 7.4 模型稳定性验证

#### 7.4.1 Bootstrap验证

**步骤**：
1. Bootstrap重采样B次
2. 每次计算模型性能
3. 计算性能的均值和置信区间

**评估**：
- 置信区间宽度（稳定性）
- 偏差（无偏性）

#### 7.4.2 交叉验证

**K折CV**：
$$CV_{error} = \frac{1}{K}\sum_{k=1}^K MSE_k$$

**留一法（LOO-CV）**：
$$CV_{LOO} = \frac{1}{n}\sum_{i=1}^n (y_i - \hat{y}_{(i)})^2$$

## 八、统计问题处理

### 8.1 多重共线性处理

**问题识别**：
- VIF > 10
- 相关系数 > 0.8
- 条件数 > 30

**解决方案**：

**方案1：变量删除**
```
1. 计算所有变量的VIF
2. 删除VIF最大的变量
3. 重新计算VIF
4. 重复直到所有VIF < 10
```

**方案2：主成分回归（PCR）**
```
1. 对X进行PCA
2. 选择前k个主成分（累计方差≥85%）
3. 用主成分作为自变量回归
```

**方案3：正则化**
- 岭回归：保留所有变量，压缩系数
- Lasso：自动变量选择

### 8.2 异方差处理

**问题识别**：
- Breusch-Pagan检验 p < 0.05
- 残差图呈喇叭形

**解决方案**：

**方案1：变换因变量**
- 对数变换：$\log(y)$
- Box-Cox变换

**方案2：加权最小二乘（WLS）**
$$\min_{\beta} \sum_{i=1}^n w_i(y_i - X_i\beta)^2$$

权重：$w_i = 1/\hat{\sigma}_i^2$

**方案3：稳健标准误**
- White标准误
- HC3标准误

### 8.3 自相关处理

**问题识别**：
- Durbin-Watson检验
- ACF/PACF图

**解决方案**：

**方案1：差分**
$$\Delta y_t = y_t - y_{t-1}$$

**方案2：AR模型**
$$y_t = \phi_1 y_{t-1} + \cdots + \phi_p y_{t-p} + X_t\beta + \epsilon_t$$

**方案3：Cochrane-Orcutt变换**

### 8.4 样本不平衡处理

**问题**：分类问题中类别不平衡

**解决方案**：

**方案1：重采样**
- 过采样（SMOTE）
- 欠采样
- 混合采样

**方案2：类别权重**
$$w_i = \frac{n}{k \cdot n_i}$$

**方案3：集成方法**
- BalancedRandomForest
- EasyEnsemble

### 8.5 缺失数据处理

**MCAR/MAR/MNAR判断**：
- Little's MCAR检验
- 缺失模式分析

**高级插补方法**：

**MICE（多重插补）**：
```
1. 对每个缺失变量建立回归模型
2. 迭代插补直到收敛
3. 生成m个完整数据集
4. 分别分析，合并结果
```

**KNN插补**：
- 找到k个最近邻
- 用邻居均值插补

### 9.1 统计模型 → 优化模型

#### 9.1.1 参数估计

**统计模型提供**：
1. **期望库存预测**：$\hat{I}_i^* = f_{stat}(X_i)$

3. **调度成本估计**：$\hat{c}_{ij} = g_{stat}(d_{ij}, \text{features})$

- 将$\hat{I}_i^*$作为目标库存

3. 分析解的鲁棒性
```

$$\min_{x} E_{\xi}[f(x, \xi)]$$
$$\text{s.t. } P(g(x, \xi) \leq 0) \geq 1-\alpha$$

### 9.2 优化模型 → 统计模型

#### 9.2.1 反馈学习

```

**强化学习框架**：
